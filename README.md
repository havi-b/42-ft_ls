# 42-ft_ls
42 Project Aimed at re-creating own version of the UNIX system command ls

### Basic functions:
* -l, -R, -a, -r, -t

### Bonus:
* -1, Dynamic file name and column sizing

### *Highlights*
* Use of recursive dynamic data tree population, sorting and printing
